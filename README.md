# Async_JS_Fetch_API

Проект + тестовый сервер для регистрации/логина с JWT

## Быстрый старт (front + demo back)

1) Установите зависимости для фронтенда и сервера:

```bash
npm install
cd server
npm install
cd ..
```

2) Запустите тестовый сервер (в отдельном окне терминала):

```bash
cd server
npm start
```

3) В другом терминале запустите фронтенд:

```bash
npm run dev
```

4) Откройте http://localhost:5173 — формы `Register`/`Login` будут работать и возвращать JWT.

> Примечание: сервер использует in-memory хранилище пользователей (демо). Для продакшена замените на реальную БД и храните секреты безопасно.
